The promoted releases are the stable releases (named 0.EVEN or 0.EVEN.N). They are all listed on the downloads page.

There are also not promoted releases called Development Releases (named 0.ODD.N, 0.EVEN.alphaN, or 0.EVEN.betaN). These are used during the development of ArgoUML. You are welcome to test these for the purpose of helping in the development or get a preview of upcoming features but there is a risk that they are not working as well as the stable releases. You can find them by following the link to the Development Releases on the downloads page.

For Developers
If you want to build ArgoUML from source, the recommended way is to download from the subversion repository. Please refer to the Developer Wiki for instructions.
If you just want to have a look at the source, you can either use the web site to browse the source or download one of these packages.
ArgoUML-0.35.1-src.tar.gz ArgoUML source code without libraries.
ArgoUML-0.35.1-src.zip ArgoUML source code without libraries.
ArgoUML-0.35.1-libs.tar.gz Libraries needed to compile ArgoUML.
ArgoUML-0.35.1-libs.zip Libraries needed to compile ArgoUML.
